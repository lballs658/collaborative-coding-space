# Auto Lab Solutions website

This is a Lovable/TanStack/Vite React project.

## Netlify settings

Build command:

```text
vite build
```

Publish directory:

```text
dist
```

The project includes `@netlify/vite-plugin-tanstack-start` and uses Nitro's `netlify` preset so the build creates the correct Netlify output.

## Form submissions

The callback form is configured as a Netlify form named `callback-request`.
After deployment, open Netlify → Site → Forms and confirm submissions appear.
Add a form notification inside Netlify if you want alerts by email.
